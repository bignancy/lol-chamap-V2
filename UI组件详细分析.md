# LOL ARAM (极地大乱斗) 选英雄页面 — UI 组件详细分析

> 基于截图 `选英雄页面.png` 的逐像素级 UI 拆解文档。  
> 用途：指导前端仿写还原，覆盖布局、颜色、渐变、边框、间距、字体等所有可视元素。

---

## 0. 全局主题与设计语言

| 属性 | 值 |
|------|-----|
| 整体背景色 | 深海军蓝 `#0A1628` ~ `#0E1A2B` |
| 主强调色 | 金色 `#C8AA6E`（边框、描边、装饰线） |
| 次强调色 | 浅金 `#F0E6D2`（文字高亮） |
| 蓝方代表色 | 蓝色 `#0A93E6` / `#0AC8E6` |
| 红方代表色 | 红色 `#E64545` / `#E6324A` |
| 文字主色 | 浅灰白 `#A09B8C`（普通文字）、`#F0E6D2`（标题/高亮） |
| 面板背景 | 半透明深蓝黑 `rgba(10, 20, 35, 0.92)` |
| 边框样式 | 金色 1px 描边 + 偶尔双层边框（外金内深色），边角有斜切/菱形裁切 |
| 字体风格 | 中文：粗体衬线/黑体；英文/数字：无衬线粗体 |
| 设计语言 | 暗色电竞风，金属质感，几何切割，斜角边框 |

## 1. 顶部标题栏（Header Bar）

### 布局
- 位于页面最顶部，占满全宽
- 高度约 **40-44px**

### 组成元素

| 元素 | 描述 | 样式细节 |
|------|------|----------|
| 背景 | 深蓝渐变 | 从 `#0A1628` 到 `#1E2D4A`，带微妙的金属光泽纹理 |
| 标题文字 | "英雄联盟" | 白色/浅金 `#F0E6D2`，居中偏左，字号约 16-18px，加粗 |
| 副标题 | "艾欧尼亚 - 经典模式" | 灰色小字，位于标题下方或右侧，字号约 11-12px |
| 底部装饰线 | 金色细线 | `#C8AA6E`，1px 高，贯穿全宽 |
| 窗口控制按钮 | 最小化 / 最大化 / 关闭 | 右上角三个图标按钮，深色背景，hover 时关闭按钮变红 |

### 边框
- 底部：1px 金色 `#C8AA6E` 实线
- 无顶部/左右边框（贴合浏览器边缘）

## 2. 倒计时器（Timer）

### 布局
- 位于标题栏下方，英雄池上方，接近页面中心偏上
- 独立浮层或嵌在英雄池容器顶部

### 样式

| 属性 | 值 |
|------|-----|
| 显示内容 | "0:23"（当前截图状态） |
| 字号 | 约 28-36px |
| 字色 | 白色 `#FFFFFF`，加粗 |
| 背景 | 无独立背景或带半透明深色圆角底板 |
| 可能的装饰 | 左右两侧可能有小的沙漏/时钟图标 |

## 3. 掷骰子 / Reroll 区域

### 布局
- 位于倒计时器左侧，英雄池容器的左上角外侧
- 水平排列

### 组成元素

| 元素 | 描述 | 样式细节 |
|------|------|----------|
| 骰子图标 | 一个小骰子图标 | 蓝色调，约 16x16px |
| 文字 | "骰子: 253 蓝色精萃" | 浅灰白色 `#A09B8C`，字号约 12px |
| 精萃图标 | 蓝色精萃的小图标 | 蓝色宝石/结晶造型 |
| Reroll 次数 | "2/2" | 白色加粗，显示当前可掷次数/最大次数 |
| 交互按钮 | 点掷骰子的按钮 | 蓝色主题按钮，可能有金色边框 |

## 4. 可选英雄池（Available Champion Pool）

### 布局
- 位于标题栏下方、两个队伍面板上方
- 占据页面顶部中央区域
- 宽度约为主内容区的 **60-70%**
- 高度约 **120-140px**（约 3 行英雄头像 + 间距）

### 容器样式

| 属性 | 值 |
|------|-----|
| 背景 | 深蓝黑半透明 `rgba(10, 20, 35, 0.9)` |
| 边框 | 1px 金色 `#C8AA6E` 描边 |
| 边角 | 斜切角（chamfered corners / diamond cut），非圆角，是 LOl 风格的几何切割 |
| 内边距 | 约 8-12px |

### 英雄头像网格

| 属性 | 值 |
|------|-----|
| 排列 | 多行网格，水平排列，自动换行 |
| 头像尺寸 | 约 **48x48px** 或 **50x50px** |
| 间距 | 水平/垂直间距约 **3-4px** |
| 可用英雄 | 全彩显示，头像清晰 |
| 不可用英雄 | 灰度化（desaturated），降低亮度约 40-50% |
| 选中状态 | 金色边框高亮，约 2px 描边 `#C8AA6E` |
| Hover 状态 | 轻微亮度提升 + 可能出现 tooltip 显示英雄名称 |
| 边框 | 每个头像有 1px 深色边框，选中时变金色 |

### 英雄头像来源
- 正方形头像图，从英雄立绘裁切
- 深色边框包裹

## 5. 板凳席（Bench）

### 布局
- 位于英雄池容器下方、两个队伍面板之间的上方区域
- 横向排列

### 组成元素

| 元素 | 描述 | 样式细节 |
|------|------|----------|
| 标签文字 | "板凳席" | 灰白色小字，字号约 11-12px |
| 板凳英雄头像 | 横排小头像 | 尺寸约 **40x40px**，比英雄池头像略小 |
| 间距 | 头像间距约 4px | 无多余装饰 |
| 背景 | 可能有深色底板 | 与英雄池风格一致 |

## 6. 队伍面板（Team Panels）

### 整体布局
- 两个面板 **左右并排**，位于英雄池下方
- 左侧面板：**蓝色方**（Team 1）
- 右侧面板：**红色方**（Team 2）
- 中间可能有间距或分隔线
- 每个面板宽度约为主内容区的 **45-48%**
- 高度约 **280-320px**

### 面板容器样式

| 属性 | 值 |
|------|-----|
| 背景 | 深蓝黑半透明 `rgba(10, 20, 35, 0.88)` |
| 边框 | 1px，蓝方面板边框偏蓝色调 `#1E3A5F`，红方面板偏红色调 `#5F1E1E` |
| 边角 | 斜切角（与英雄池一致） |
| 顶部标签 | 蓝方："蓝色方" / 红方："红色方"，位于面板顶部居中 |

### 玩家槽位（每个面板 5 个）

每个槽位的布局（横向排列或纵向堆叠）：

| 元素 | 描述 | 样式细节 |
|------|------|----------|
| 槽位背景 | 深色半透明 | 未选择时较暗，选择后有英雄色彩 |
| 英雄头像 | 已选英雄的头像 | 约 **60x60px**，位于槽位左侧 |
| 玩家名称 | 召唤师名称 | 浅色文字，字号约 12-13px，位于头像右侧 |
| 召唤师技能 | 2 个技能图标 | 约 **24x24px**，小图标，位于名称下方或头像旁 |
| 符文图标 | 主要符文图标 | 约 **20x20px**，小圆形图标 |
| 锁定状态 | 已锁定标记 | 可能显示锁定图标或改变边框颜色 |
| 当前用户高亮 | 自己的槽位 | 可能有金色边框或背景高亮区分 |

### 槽位排列
- **竖向排列**：5 个槽位从上到下依次排列
- 槽位之间有 1px 分隔线或间距
- 未有英雄选择的槽位显示为空暗格

| 遮罩 | 底部有渐变遮罩，从图片过渡到下方深色背景 |
| 边框 | 面板整体有金色 1px 描边 |

#### 7b. 英雄名称区域

| 元素 | 样式 |
|------|------|
| 英雄称号 | "卡牌大师" — 金色 `#C8AA6E`，字号约 18-20px，加粗 |
| 英雄名字 | "崔斯特" — 白色 `#F0E6D2`，字号约 14-16px |
| 分隔 | 名称与技能之间有 1px 金色细线 |

#### 7c. 技能展示区域

| 属性 | 值 |
|------|-----|
| 布局 | 技能图标横向排列（被动 + Q + W + E + R） |

| 选中技能 | 金色边框高亮（截图中 R 技能 "命运" 被选中高亮） |

#### 7d. 技能详情区域

| 属性 | 值 |
|------|-----|

| 快捷键标签 | "R" 键位标签，小方框包裹 |

| 内边距 | 约 12-16px |

## 9. 底部操作栏（Footer / Action Bar）

### 布局
- 位于页面最底部，占满全宽
- 高度约 **48-54px**
- 固定在视口底部

### 背景
- 深蓝黑渐变，与顶部标题栏呼应
- 顶部有 1px 金色 `#C8AA6E` 分隔线

### 组成元素

#### 9a. 左侧按钮组

| 按钮 | 文字 | 样式 |
|------|------|------|
| 菜单按钮 | "≡ 菜单" | 深色背景按钮，浅色文字，可能有 1px 深灰边框 |
| 交换按钮 | "向队友发出交换请求" | 深色背景按钮，浅色文字，与菜单按钮风格一致 |

#### 9b. 右侧主操作按钮

| 按钮 | 文字 | 样式 |
|------|------|------|
| 确认选择 | "确认选择"（未锁定状态） | **金色渐变按钮** `#C8AA6E` → `#9B7F4C`，白色粗体文字，斜切边角，hover 时变亮 |
| 确认锁定 | "确认锁定"（已选择英雄后） | 同上样式，文字变化 |

### 按钮样式细节

```
确认选择 按钮：
- 背景: 线性渐变, 从 #C8AA6E (顶部) 到 #9B7F4C (底部)
- 文字: #FFFFFF, 加粗, 字号约 14-16px
- 边框: 1px #D4AF37 金色描边
- 边角: 斜切角 (clip-path 或 border 切割)
- 尺寸: 约 160x40px
- Hover: 渐变亮化, #D4B87A → #AD8F5C
- Active/Pressed: 渐变暗化
```

```
菜单/交换 按钮：
- 背景: #1E2328 (深灰蓝)
- 文字: #A09B8C (灰白)
- 边框: 1px #3C3C3C
- 边角: 斜切角
- 尺寸: 约 120x36px
- Hover: 背景变亮 #2E3338
```

## 12. 关键交互状态

### 英雄头像交互

| 状态 | 视觉表现 |
|------|----------|
| Default (可用) | 全彩，深色 1px 边框 |
| Default (不可用) | 灰度化，亮度 -40%，不可点击光标 |
| Hover | 亮度 +15%，可能出现 tooltip |
| Selected | 2px 金色边框 `#C8AA6E`，亮度 +10% |
| Locked (他人已选) | 被选走的英雄可能带蓝色/红色标记 |

### 确认按钮交互

| 状态 | 视觉表现 |
|------|----------|
| Default (未选英雄) | 暗色/禁用态，文字 "确认选择" |
| Active (已选英雄) | 金色渐变高亮，可点击 |
| Hover | 渐变亮化 |
| Pressed | 渐变暗化 + 微缩效果 |
| Locked | 文字变为 "确认锁定"，可能变色 |

### 槽位交互

| 状态 | 视觉表现 |
|------|----------|
| Empty | 暗色空槽，可能有虚线边框 |
| Has Champion | 英雄头像 + 名称 + 技能图标 |
| Current User | 金色边框高亮 |
| Locked | 锁定图标覆盖 |

## 15. 实现优先级建议

按截图还原时的开发顺序：
